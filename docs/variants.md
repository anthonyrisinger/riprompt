### RIP1: The Primordial Prompt
**File:** `rip/rip1.txt`

SYNTH VARIANT_NAME UNTIL SEMIOTIC GUIDE SYNTH!

**Key principles:**
1. **Repetition with variation**: Same structure, different content

4. **Emoji integration**: Use emoji as semantic primitives
5. **ALL-CAPS emphasis**: Grab attention at cold start
